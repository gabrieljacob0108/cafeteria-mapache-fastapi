## 🧩 Arquitectura e integración de módulos

Este proyecto simula un ecosistema comercial donde los módulos están conectados mediante lógica de negocio real, no solo una base de datos compartida.

- **Gestión de inventario:** el Almacén abastece de stock al Supermercado y a la Máquina Expendedora. Cuando el stock se agota, estos módulos notifican al Almacén para actualizar las existencias.

- **Sistema de recompensas:** la Cafetería, la Pastelería y el Sistema de Tokens trabajan en conjunto. Cada compra registra el monto de dinero y otorga puntos automáticamente al usuario según su historial de consumo.
